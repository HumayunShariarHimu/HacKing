1️⃣ Information

সংজ্ঞা:
Credential Harvesting হলো আক্রমণ যেখানে আক্রমণকারী ইউজারের লগইন তথ্য (username, password, token) সংগ্রহ করে, সাধারণত phishing, malware বা lab-simulated methods ব্যবহার করে।

বৈশিষ্ট্য:

লক্ষ্য: Web applications, SaaS, Cloud services, Email accounts

উদ্দেশ্য: Unauthorized access, privilege escalation, data theft

গবেষণার জন্য ব্যবহার: lab simulation, penetration testing, security research


ধাপসমূহ (Attack Lifecycle – গবেষণার দৃষ্টিকোণ)

1. Reconnaissance – Target system এবং user information collection


2. Credential Capture – lab simulation via phishing pages, keyloggers, or MITM


3. Exploitation – lab-only access using harvested credentials


4. Actions on Objectives – login to lab applications, privilege escalation, logging



টেকনিক:

Phishing page deployment in lab environment

Keylogger simulation (lab-only)

MITM / network sniffing for credentials

Session replay / token capture simulation

Credential storage / database simulation

2️⃣ Method

Lab Setup:

VirtualBox / VMware – isolated lab VM

Kali Linux / Ubuntu VM

Web application sandbox (DVWA, WebGoat)

Android Emulator (Android Studio / Genymotion)

Isolated network for credential testing


Credential Harvesting Simulation:

Deploy lab phishing page or credential capture form

Simulate user login and capture credentials

Replay harvested credentials to access lab resources

Log all credential attempts and success rate


Data Collection & Analysis:

Capture success rate

Unauthorized login attempts metrics

Statistical analysis: credential reuse, access patterns

Visualization: login attempts, credential logs


Ethical / Legal Considerations:

All credentials and accounts are lab-simulated

Licensed/open-source tools only

Sandbox / isolated network
