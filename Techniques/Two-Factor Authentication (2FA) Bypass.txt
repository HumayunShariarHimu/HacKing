1️⃣ Information

সংজ্ঞা:
Two-Factor Authentication (2FA) Bypass হলো আক্রমণ যেখানে আক্রমণকারী ব্যবহারকারীর 2FA প্রক্রিয়া এড়িয়ে গিয়ে unauthorized access পায়। এটি credential theft ছাড়াও session takeover বা social engineering ব্যবহার করে করা হয়।

বৈশিষ্ট্য:

লক্ষ্য: ওয়েব অ্যাপ্লিকেশন, SaaS, ক্লাউড সার্ভিস

Credential + 2FA token বা code ব্যবহার ছাড়া access

উদ্দেশ্য: Unauthorized access, privilege escalation, data theft

গবেষণার জন্য ব্যবহার: lab simulation, penetration testing, security research


ধাপসমূহ (Attack Lifecycle – গবেষণার দৃষ্টিকোণ)

1. Reconnaissance – টার্গেট ওয়েব অ্যাপ্লিকেশন ও 2FA mechanism পর্যবেক্ষণ


2. 2FA Bypass Technique Identification – lab simulation of bypass methods


3. Exploitation – lab environment এ bypass execution


4. Actions on Objectives – session takeover, resource access, logging



টেকনিক:

Social engineering / phishing for token capture

SIM swap simulation (lab-only)

Exploiting predictable 2FA codes (TOTP/HOTP) in lab environment

Session token replay / reuse simulation

Backup code exploitation simulation

2️⃣ Method

Lab Setup:

Deploy lab test account with TOTP / SMS 2FA

Simulate phishing / social engineering to capture 2FA token in lab

Test backup codes, token replay, or session hijacking methods

Log access attempts, bypass success, and system responses


Data Collection & Analysis:

Bypass success metrics

Session takeover and privilege escalation logs

Statistical analysis: code reuse, token prediction, user response rate

Visualization: 2FA bypass timeline, access logs


Ethical / Legal Considerations:

Licensed/open-source tools only
