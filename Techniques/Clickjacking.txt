1️⃣ Information

সংজ্ঞা:
Clickjacking হলো আক্রমণ যেখানে আক্রমণকারী ব্যবহারকারীর ক্লিককে অন্য কার্যকলাপে reroute করে, যাতে ব্যবহারকারী অজান্তে ক্ষতিকারক কার্য সম্পাদন করে। এটি সাধারণত iframe overlay বা CSS trickery ব্যবহার করে করা হয়।

বৈশিষ্ট্য:

লক্ষ্য: Web applications, buttons, links, forms

উদ্দেশ্য: Unauthorized actions, privilege escalation, lab-only simulation of malicious clicks

গবেষণার জন্য ব্যবহার: lab simulation, penetration testing, security research


ধাপসমূহ (Attack Lifecycle – গবেষণার দৃষ্টিকোণ)

1. Reconnaissance – Identify target buttons, links, or UI elements


2. Payload Crafting – lab-only overlay page creation


3. Exploitation – Embed target page in hidden iframe / CSS overlay


4. Actions on Objectives – Capture or simulate unintended user clicks


5. Post-Exploitation – Log click simulation, evaluate lab impact



টেকনিক:

Hidden iframe overlay on lab test page

CSS opacity / z-index manipulation

Mouse event capturing simulation

Redirection via lab-controlled pages

JavaScript event hijacking in lab environment

2️⃣ Method

Lab Setup:

VirtualBox / VMware – isolated lab VM

Kali Linux / Ubuntu VM

Web application sandbox (lab HTML/JS pages for clickjacking)

Isolated network for lab testing


Clickjacking Simulation:

Create lab test page with sensitive buttons or forms

Overlay iframe with transparency over lab button

Capture clicks in lab environment without affecting real users

Log event simulation and evaluate exploit effectiveness


Data Collection & Analysis:

Click capture success metrics in lab

Simulation of unintended actions executed

Statistical analysis: user interaction simulation, overlay effectiveness

Visualization: click paths, overlay impact


Ethical / Legal Considerations:

All targets and users are lab-simulated

Licensed/open-source tools only

Sandbox / isolated network
