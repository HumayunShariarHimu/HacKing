1️⃣ Information

সংজ্ঞা:
Baiting হলো সোশ্যাল ইঞ্জিনিয়ারিং আক্রমণের একটি ধরণ, যেখানে আক্রমণকারী লোভনীয় অফার, ফ্রি ডাউনলোড, বা প্রলোভনমূলক সামগ্রী ব্যবহার করে ভিকটিমকে ব্যক্তিগত তথ্য, credential বা ম্যালওয়্যার ইনস্টল করতে প্ররোচিত করে।

বৈশিষ্ট্য:

লোভনীয় প্রলোভন (Free USB, download, coupons, gifts)

ডেলিভারি চ্যানেল: ফিজিক্যাল মিডিয়া (USB, CD) অথবা অনলাইন (Download links, Ads)

উদ্দেশ্য: Credential theft, malware installation, sensitive data exfiltration

গবেষণার জন্য ব্যবহার: সিমুলেশন, ট্রেনিং, এডুকেশনাল প্র্যাকটিস


ধাপসমূহ (Attack Lifecycle – গবেষণার দৃষ্টিকোণ)

1. Reconnaissance – টার্গেট সম্পর্কিত তথ্য সংগ্রহ


2. Bait Creation – লোভনীয় অফার বা ফ্রি সামগ্রী তৈরি


3. Delivery – USB, CD, বা Online download link (ল্যাবে)


4. Exploitation – ভিকটিম ফাইল খুললে ম্যালওয়্যার বা credential logging (সিমুলেটেড)


5. Actions on Objectives – ডেটা লগিং, ক্লিক রেট, response tracking



টেকনিক:

Physical baiting (USB, CD, storage devices)

Online baiting (download links, ads, fake offers)

File embedding (malicious script simulation in lab)

Social engineering scripts for persuasion

Personalization based on OSINT

2️⃣ Method

Lab Setup:

VirtualBox / VMware – isolated lab VM

Kali Linux / Ubuntu VM

Android Emulator (Android Studio / Genymotion)

Isolated network for download or USB simulation


Baiting Campaign Simulation:

Create simulated “free offer” files or links

Monitor lab VM for interaction (file open, link click)

Log user responses & simulated malware execution

Statistical analysis: click rate, interaction rate


Data Collection & Analysis:

Interaction success rate

User response metrics

Visualization: time-based interaction trends, heatmaps


Ethical / Legal Considerations:

Target data সম্পূর্ণ সিমুলেটেড

Only licensed/open-source tools

Network isolation / sandboxing
