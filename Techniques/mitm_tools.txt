Man-in-the-Middle (MITM) আক্রমণ পরিচালনার জন্য প্রয়োজনীয় বিভিন্ন টুলস, প্যাকেজ, কম্পিউটার সফটওয়্যার, এবং হার্ডওয়্যার ব্যবহার করা হয়। এই টুলগুলি কেবল শিক্ষণীয় উদ্দেশ্যে এবং পরীক্ষা-নিরীক্ষা পরিবেশে ব্যবহৃত হওয়া উচিত। বৈধ এবং নিরাপদ ব্যবহারের জন্য এগুলি ব্যবহার করুন এবং কখনো অনৈতিকভাবে ব্যবহার করবেন না। নিচে কিছু প্রয়োজনীয় টুল এবং উপকরণের তালিকা দেওয়া হল:


---

1. কম্পিউটার সফটওয়্যার এবং টুলস

a. Ettercap

বর্ণনা: Ettercap একটি শক্তিশালী MITM টুল যা ARP স্পুফিং, DNS স্পুফিং, প্যাকেট স্নিফিং, এবং অন্যান্য আক্রমণগুলির জন্য ব্যবহৃত হয়।

ইনস্টলেশন (Linux):

sudo apt-get install ettercap-graphical


b. Wireshark

বর্ণনা: Wireshark একটি জনপ্রিয় নেটওয়ার্ক স্নিফার টুল যা প্যাকেট ক্যাপচার এবং বিশ্লেষণের জন্য ব্যবহৃত হয়। এটি MITM আক্রমণের সময় ট্রাফিক বিশ্লেষণে সাহায্য করে।

ইনস্টলেশন (Linux):

sudo apt-get install wireshark


c. MITMf (Man-in-the-Middle Framework)

বর্ণনা: MITMf একটি অত্যন্ত জনপ্রিয় এবং উন্নত MITM আক্রমণ ফ্রেমওয়ার্ক যা একাধিক আক্রমণ কৌশল সরবরাহ করে, যেমন DNS spoofing, SSL stripping, session hijacking, etc.

ইনস্টলেশন:

git clone https://github.com/byt3bl33d3r/MITMf.git
cd MITMf
sudo python3 setup.py install


d. SSLstrip

বর্ণনা: SSLstrip একটি টুল যা HTTPS ট্রাফিককে HTTP তে রিডাইরেক্ট করে, ফলে SSL এনক্রিপশন বাইপাস করা হয় এবং আক্রমণকারী প্লেইনটেক্সট ট্রাফিক পায়।

ইনস্টলেশন:

sudo apt-get install sslstrip


e. Dsniff

বর্ণনা: Dsniff একটি টুলসেট যা প্যাকেট স্নিফিং এবং বিভিন্ন MITM আক্রমণের জন্য ব্যবহৃত হয়, যেমন DNS Spoofing, SMTP, FTP স্নিফিং ইত্যাদি।

ইনস্টলেশন (Linux):

sudo apt-get install dsniff


f. Aircrack-ng

বর্ণনা: Aircrack-ng একটি জনপ্রিয় ওয়াই-ফাই হ্যাকিং টুলসেট যা Evil Twin attack এবং WEP/WPA/WPA2 পাসওয়ার্ড ক্র্যাকিং-এর জন্য ব্যবহৃত হয়।

ইনস্টলেশন:

sudo apt-get install aircrack-ng


g. Cain and Abel

বর্ণনা: Cain and Abel Windows-এ ব্যবহৃত একটি শক্তিশালী পাসওয়ার্ড রিকভারি টুল যা ARP স্পুফিং, DNS স্পুফিং, এবং প্যাকেট স্নিফিং করার জন্য ব্যবহার করা যায়।

ডাউনলোড: Cain and Abel Official Website

2. এনড্রোয়েড টুলস

a. zANTI

বর্ণনা: zANTI একটি শক্তিশালী Android হ্যাকিং টুল যা MITM আক্রমণ করার জন্য ব্যবহৃত হয়। এটি ARP Spoofing, DNS Spoofing, এবং প্যাকেট স্নিফিং সমর্থন করে।

ডাউনলোড: zANTI Official Website

প্রয়োজনীয়তা: Rooted Android ডিভাইস


b. NetCut

বর্ণনা: NetCut একটি Android অ্যাপ্লিকেশন যা নেটওয়ার্ক ম্যানেজমেন্ট এবং ARP Spoofing আক্রমণ করার জন্য ব্যবহৃত হয়।

ডাউনলোড: NetCut Android App


c. Intercepter-NG

বর্ণনা: Intercepter-NG একটি Android অ্যাপ যা Wi-Fi নেটওয়ার্কে MITM আক্রমণ করতে ব্যবহৃত হয়, বিশেষত প্যাকেট স্নিফিং এবং পাসওয়ার্ড চুরি করতে।

ডাউনলোড: Intercepter-NG


d. Wi-Fi Kill

বর্ণনা: Wi-Fi Kill একটি Android অ্যাপ যা Wi-Fi নেটওয়ার্কে থাকা অন্যান্য ব্যবহারকারীদের সংযোগ বিচ্ছিন্ন করতে এবং MITM আক্রমণ পরিচালনা করতে সক্ষম।

ডাউনলোড: Wi-Fi Kill on GitHub

3. হার্ডওয়্যার

a. Wi-Fi Pineapple

বর্ণনা: Wi-Fi Pineapple একটি অত্যন্ত শক্তিশালী পেনিট্রেশন টেস্টিং ডিভাইস যা বিশেষভাবে MITM আক্রমণ পরিচালনা করার জন্য ডিজাইন করা হয়েছে। এটি Evil Twin Attack, DNS Spoofing, এবং Man-in-the-Middle আক্রমণের জন্য ব্যবহার করা হয়।

বিশেষত্ব: এটি অনেক প্রি-কনফিগারড আক্রমণ টুল এবং স্ক্রিপ্ট সরবরাহ করে।


b. USB Rubber Ducky

বর্ণনা: USB Rubber Ducky একটি পোর্টেবল ডিভাইস যা কীবোর্ড ইমুলেটর হিসেবে কাজ করে এবং আক্রমণকারীকে শেল স্ক্রিপ্ট রান করতে সক্ষম করে, ফলে এটি MITM আক্রমণে সাহায্য করতে পারে।

ডাউনলোড: USB Rubber Ducky Official Website


c. Alfa AWUS036NHA (Wi-Fi Adapter)

বর্ণনা: Alfa AWUS036NHA একটি শক্তিশালী ওয়াই-ফাই অ্যাডাপ্টার যা অ্যালাইড কনফিগারেশন এবং Wi-Fi প্যাকেট স্নিফিংয়ের জন্য ব্যবহৃত হয়। এটি Wi-Fi আক্রমণের জন্য আদর্শ।

বিশেষত্ব: এটি নেটওয়ার্কের ডেটা সংগ্রহ এবং MITM আক্রমণ করার জন্য সহায়ক।

4. নেটওয়ার্ক ও ফায়ারওয়াল টুলস

a. Tcpdump

বর্ণনা: Tcpdump একটি কমান্ড-লাইন নেটওয়ার্ক স্নিফার টুল যা প্যাকেট ক্যাপচার এবং বিশ্লেষণ করতে ব্যবহৃত হয়। এটি MITM আক্রমণের সময় প্যাকেট স্নিফিংয়ের জন্য ব্যবহৃত হয়।

ইনস্টলেশন (Linux):

sudo apt-get install tcpdump


b. Burp Suite

বর্ণনা: Burp Suite একটি জনপ্রিয় ওয়েব অ্যাপ্লিকেশন সিকিউরিটি টেস্টিং টুল, যা MITM আক্রমণগুলি পরীক্ষা করার জন্য ব্যবহৃত হয়। এটি প্যাকেট স্নিফিং, SSL/TLS ইনজেকশন, এবং অন্যান্য অনেক অ্যাটাক টেস্ট করতে সাহায্য করে।

ডাউনলোড: Burp Suite Official Website

উপসংহার:

MITM আক্রমণ পরিচালনার জন্য উপরের টুলগুলি একটি শক্তিশালী সেট তৈরি করে, তবে মনে রাখবেন যে এই টুলগুলি সেগুলি শুধু শিক্ষণীয় বা পেনিট্রেশন টেস্টিং পরিবেশে ব্যবহার করা উচিত। বৈধ অনুমতি ছাড়া এই টুলগুলি ব্যবহার করা বেআইনি এবং গুরুতর আইনি ফলাফল ডেকে আনতে পারে।
