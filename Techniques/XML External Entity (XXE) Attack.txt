1️⃣ Information

সংজ্ঞা:
XML External Entity (XXE) Attack হলো আক্রমণ যেখানে আক্রমণকারী একটি ওয়েব অ্যাপ্লিকেশন বা সার্ভার এর XML পার্সিং দুর্বলতা ব্যবহার করে sensitive data access, SSRF, Denial-of-Service (DoS), বা remote code execution করতে পারে।

বৈশিষ্ট্য:

লক্ষ্য: Web applications, APIs, SOAP/XML endpoints

মূল দুর্বলতা: Unsafe XML parsing with external entity processing enabled

উদ্দেশ্য: Sensitive file disclosure, internal network scanning, denial-of-service, lab simulation of RCE

গবেষণার জন্য ব্যবহার: lab simulation, penetration testing, security research


ধাপসমূহ (Attack Lifecycle – গবেষণার দৃষ্টিকোণ)

1. Reconnaissance – Identify XML endpoints and parsing libraries


2. Payload Crafting – lab-only external entity payload creation


3. Exploitation – Inject payload into XML input fields in lab environment


4. Actions on Objectives – Access local files, simulate SSRF, or DoS in lab


5. Post-Exploitation – Log responses, evaluate exploit impact



টেকনিক:

External Entity Injection (ENTITY declarations)

Local File Inclusion via XML parser

Out-of-Band (OOB) interactions for lab simulation

XXE blind attacks (lab environment only)

DoS / Billion Laughs / Entity Expansion simulation

Deploy lab web app or API endpoint that accepts XML

Identify vulnerable XML parser (lab environment)

Craft XML payload with external entity referencing lab files or internal resources

Send payload to lab server, capture server response

Log file disclosure, SSRF simulation, or DoS impact


Data Collection & Analysis:
