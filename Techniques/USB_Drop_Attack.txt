1️⃣ Information

সংজ্ঞা:
USB Drop Attack হলো সোশ্যাল ইঞ্জিনিয়ারিং আক্রমণের একটি ধরণ, যেখানে ক্ষতিকারক ইউএসবি ডিভাইস ভিকটিমের কাছে ছেড়ে দেওয়া হয়। ভিকটিম যখন ডিভাইসটি কম্পিউটারে সংযুক্ত করে, তখন ম্যালওয়্যার ইনস্টল হয় বা credential চুরি হয়।

বৈশিষ্ট্য:

লোভনীয় বা নির্দিষ্ট ইউএসবি ডিভাইস ব্যবহার

ভিকটিমের হাতে ডিভাইস ছেড়ে দেওয়া

উদ্দেশ্য: Credential theft, malware installation, sensitive data exfiltration

গবেষণার জন্য ব্যবহার: সিমুলেশন, ট্রেনিং, এডুকেশনাল প্র্যাকটিস


ধাপসমূহ (Attack Lifecycle – গবেষণার দৃষ্টিকোণ)

1. Reconnaissance – টার্গেট সিস্টেম ও পরিবেশের তথ্য সংগ্রহ


2. Payload Creation – সিমুলেটেড ক্ষতিকারক ফাইল বা স্ক্রিপ্ট তৈরি


3. Delivery – USB drop simulation in lab


4. Exploitation – ভিকটিম ডিভাইসটি খোলার সাথে সাথে ল্যাবে সিমুলেটেড payload কার্যকর


5. Actions on Objectives – লগিং, credential collection, interaction tracking



টেকনিক:

Malicious file embedding (simulated scripts)

Auto-run / Auto-execute simulation (lab-only)

Social engineering pretexting to encourage plug-in

Data exfiltration simulation

USB device impersonation (keyboard/mouse attack simulation in lab)

2️⃣ Method

Lab Setup:

VirtualBox / VMware – isolated lab VM

Kali Linux / Ubuntu VM

Android Emulator (Android Studio / Genymotion)

Isolated network for lab simulation


USB Drop Simulation:

Create simulated USB payload (benign scripts for lab)

Attach to lab VM or emulator

Log file execution, access attempts, and system responses

Analyze interaction and simulated malware effects


Data Collection & Analysis:

Interaction success rate

Script execution metrics

Statistical analysis: response rate, execution attempts

Visualization: timeline of interaction, logs


Ethical / Legal Considerations:

All USB drops are lab-simulated

Licensed/open-source tools only

Sandbox / network isolation
