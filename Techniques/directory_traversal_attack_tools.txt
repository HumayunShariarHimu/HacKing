Directory Traversal Attack করার জন্য প্রয়োজনীয় টুলস, প্যাকেজ, সফটওয়্যার ও হার্ডওয়্যার:

এনড্রোয়েড টুলস:

1. Termux:

বর্ণনা: Termux হল একটি টার্মিনাল এমুলেটর যা প্যাকেজ ব্যবস্থাপক ব্যবহার করে বিভিন্ন সিকিউরিটি টুলস ইনস্টল করতে সহায়তা করে।

প্যাকেজ: আপনি Termux ব্যবহার করে বিভিন্ন টুল ইনস্টল করতে পারবেন, যেমন nmap, curl, wget, বা Python স্ক্রিপ্ট।



2. DroidBox:

বর্ণনা: DroidBox একটি প্যাকেজ বিশ্লেষণ টুল যা অ্যাপ্লিকেশন ফাইলের নিরাপত্তা দুর্বলতা পরীক্ষা করতে ব্যবহৃত হয়। এটি Directory Traversal অ্যাটাক সনাক্ত করতে সহায়তা করতে পারে।



3. zANTI:

বর্ণনা: zANTI একটি পোর্টেবল পেনটেস্টিং টুল, যা বিভিন্ন নেটওয়ার্ক নিরাপত্তা পরীক্ষা করতে ব্যবহৃত হয়। এতে Directory Traversal-এর সনাক্তকরণও করা সম্ভব।




কম্পিউটার সফটওয়্যার:

1. Burp Suite:

বর্ণনা: Burp Suite একটি ওয়েব অ্যাপ্লিকেশন সিকিউরিটি টেস্টিং টুল। এটি ইনপুট প্যারামিটার বিশ্লেষণ করতে ব্যবহৃত হয় এবং আক্রমণকারী প্যারামিটারগুলিতে ডিরেক্টরি ট্র্যাভার্সাল ইনজেক্ট করতে সক্ষম।

লিঙ্ক: https://portswigger.net/burp



2. OWASP ZAP (Zed Attack Proxy):

বর্ণনা: OWASP ZAP একটি জনপ্রিয় ওপেন সোর্স ওয়েব অ্যাপ্লিকেশন নিরাপত্তা স্ক্যানার যা Directory Traversal এবং অন্যান্য আক্রমণ শনাক্ত করতে সহায়তা করে।



3. Nikto:

বর্ণনা: Nikto একটি ওপেন সোর্স ওয়েব সার্ভার স্ক্যানার যা বিভিন্ন ধরনের নিরাপত্তা দুর্বলতা খুঁজে বের করতে সক্ষম। এটি ওয়েব অ্যাপ্লিকেশন ভ্যালিডেশন পরীক্ষা করতে ব্যবহৃত হয়।



4. Metasploit Framework:

বর্ণনা: Metasploit হল একটি পেনটেস্টিং টুল যা বিভিন্ন ধরনের আক্রমণ (যেমন Directory Traversal) পরিচালনা করার জন্য ব্যবহৃত হয়।

লিঙ্ক: https://www.metasploit.com/



5. DirBuster:

বর্ণনা: DirBuster একটি ব্রুটফোর্স টুল যা ওয়েব সার্ভারের বিভিন্ন ডিরেক্টরি এবং ফাইল খুঁজে বের করতে ব্যবহৃত হয়। এটি ডিরেক্টরি ট্র্যাভার্সাল আক্রমণের সময় ফাইল সিস্টেমের আক্রমণ করতে সাহায্য করে।



6. Hydra:

বর্ণনা: Hydra একটি পাসওয়ার্ড ক্র্যাকিং টুল, যা Directory Traversal আক্রমণ করতে পারে যেখানে ইউজারনেম এবং পাসওয়ার্ডের জন্য সঠিক ইনপুটের চেষ্টা করা হয়।



7. Wfuzz:

বর্ণনা: Wfuzz একটি ওয়েব অ্যাপ্লিকেশন ফাজিং টুল যা বিভিন্ন প্যারামিটার পরীক্ষা করে এবং ডিরেক্টরি ট্র্যাভার্সাল আক্রমণ সনাক্ত করতে সহায়তা করে।



8. SQLmap:

বর্ণনা: SQLmap একটি শক্তিশালী টুল যা SQL ইনজেকশন দুর্বলতা পরীক্ষা করতে ব্যবহৃত হয়, তবে এটি Directory Traversal আক্রমণের সনাক্তকরণে সাহায্য করতে পারে যখন এটি ওয়েব সার্ভারের ফাইল সিস্টেম অ্যাক্সেস করতে সক্ষম হয়।




হার্ডওয়্যার:

1. Wi-Fi Adapter (এথারনেট/ওয়াইফাই) - Alfa AWUS036NHA:

বর্ণনা: এটি একটি শক্তিশালী ওয়াইফাই অ্যাডাপ্টার যা পেনটেস্টিং ও নেটওয়ার্ক স্ক্যানিংয়ের জন্য ব্যবহৃত হয়। এটি নেটওয়ার্কের দুর্বলতা স্ক্যান করতে সাহায্য করতে পারে এবং যদি সিস্টেমের মধ্যে ট্র্যাভার্সাল আক্রমণ পরীক্ষা করা হয়, তবে নেটওয়ার্ক অ্যাক্সেস করতে পারে।



2. Raspberry Pi (পেনটেস্টিং ডিভাইস):

বর্ণনা: Raspberry Pi হল একটি সস্তা কিন্তু শক্তিশালী কম্পিউটার, যা পেনটেস্টিং টুলস চালাতে ব্যবহৃত হয়। এর মাধ্যমে আপনি ওয়েব সার্ভারের বিভিন্ন দুর্বলতা পরীক্ষা করতে পারেন।



3. USB Rubber Ducky:

বর্ণনা: USB Rubber Ducky একটি কিবোর্ড এমুলেটিং ডিভাইস, যা বিশেষভাবে পেনটেস্টিংয়ের জন্য ব্যবহৃত হয়। এটি আক্রমণকারী স্ক্রিপ্ট ইনজেক্ট করতে ব্যবহার করা যেতে পারে।

এই টুলস, সফটওয়্যার এবং হার্ডওয়্যার আপনার Directory Traversal আক্রমণ প্র্যাকটিস করতে সহায়ক হবে এবং সিস্টেমের দুর্বলতা খুঁজে বের করার জন্য উপযোগী হবে। এটি একটি সঠিক পরিবেশে ব্যবহার করুন, যেমন একটি আইনি এবং শুদ্ধ উদ্দেশ্যে নিরাপত্তা প্রশিক্ষণের জন্য অথবা ওয়েব অ্যাপ্লিকেশনের সুরক্ষা বৃদ্ধির জন্য।
